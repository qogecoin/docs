BenchmarkingÂ¶

Qogecoin Core has an internal benchmarking framework, with benchmarks for cryptographic algorithms such as SHA1, SHA256, SHA512 and RIPEMD160. As well as the rolling bloom filter.

After compiling qogecoin-core, the benchmarks can be run with:

src/bench/bench_qogecoin

The output will look similar to:
